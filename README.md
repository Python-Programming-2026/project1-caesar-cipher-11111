## 项目介绍
这是一个使用 Python 和 Tkinter 实现的凯撒密码加密/解密图形界面工具。凯撒密码是一种经典的替换密码，通过将字母按照固定的移位量进行循环移位来实现加密和解密。本工具提供了直观的窗口界面，用户可输入文本、指定移位量，一键完成加密或解密操作，并支持复制结果到剪贴板。
## 程序部署过程
1.确保本地计算机已安装Python以及jupyter。  
2.将本项目代码保存为一个.ipynb文件。  
3.运行该文件。  
## 功能特点
·对英文字母用凯撒密码进行加密与解密，非字母字符原样保留  
·可交互界面  
·对于输入错误可以进行提示  
## 使用环境
操作系统：Windows  
Python 版本：3.8  
Anaconda中的jupyter  
## 使用方法
1.运行程序后，弹出主窗口  
2.在“输入文本”框中键入要处理的文本  
3.在“移位量（整数）”框中输入一个整数  
4.选择“加密”或“解密”模式  
5.点击“执行”按钮，结果将显示在下方的“结果”框中
